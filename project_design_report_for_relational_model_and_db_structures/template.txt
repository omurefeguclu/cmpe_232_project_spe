II.	Relational Model 

a.	Definition: 
In this section, provide a concise yet comprehensive explanation of the relational model employed in your project. The relational model is a conceptual framework used to organize and represent data in a relational database management system (RDBMS). It consists of tables, where each table represents an entity, and columns within the table represent attributes of that entity. Relationships between tables are established using keys, such as primary keys and foreign keys, to maintain data integrity and enforce referential integrity constraints.
